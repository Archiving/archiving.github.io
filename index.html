About Me

I first started programming while in high school, being inspired to start through the development of modifications to games, mostly in Java, adding new content in order to make the games more fun for me and friends. From this, I moved on to persuing more serious programming projects, including both software and game development with Java/C++. I now have experience in multiple game development/graphics libraries, most well known being experience with the Unity3D engine, OpenGL 3D game development, and experience in LWJGL (Lightweight Java Game Library) for 2D and 3D game development.
